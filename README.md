# Financial Fortress Dashboard

The **Financial Fortress Dashboard** is a serverless, client-side web application designed to help families and solopreneurs build economic resilience. By leveraging your browser's `localStorage`, this tool provides a private, secure, and offline-capable workstation for managing your personal finances and household efficiency.

## Overview
This dashboard acts as a digital implementation of the defensive financial strategies outlined in the **Recession-Proof Your Life: Essential Tips for Families and Entrepreneurs** eBook. It includes tools for:
* **50/30/20 Budgeting:** Real-time allocation of income into Survival, Comforts, and Fortification tiers.
* **30-Day Spending Freeze:** A behavioral tool to track and curb impulse purchases.
* **Vampire Power & Household Audit:** A checklist to identify and eliminate utility and energy waste.

## Features
* **Glassmorphism Design:** A modern, Nordic-inspired minimalist aesthetic.
* **Data Visualization:** Interactive donut charts to visualize your budget tiers.
* **Local Data Persistence:** All data is stored in your browser's `localStorage`—no server-side databases required.
* **Privacy-First:** Your financial data never leaves your device.
* **Responsive:** Optimized for both desktop and mobile use, perfect for off-grid or on-the-go planning.

## How to Deploy
1.  Fork this repository to your GitHub account.
2.  Enable **GitHub Pages** in the repository **Settings > Pages**.
3.  Your live dashboard will be available at `https://[your-username].github.io/[your-repo-name]/`.

## Privacy & Security
This application is "serverless." No account creation or login is required. Because data is stored locally in your browser, please ensure you use a private device and clear your browser cache if you share the machine. For added security, use the "Download Summary" feature to save a local `.txt` backup of your financial data.
